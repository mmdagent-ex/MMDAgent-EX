# MMDAgent-EX へのコントリビューション（日本語）

MMDAgent-EX は、研究・教育・実験用途での利用を想定したオープンソースの対話エージェント基盤です。小さな報告や実験結果の共有も、重要な貢献になります。

---

## どのような貢献を歓迎していますか？

以下のような形の貢献を歓迎しています。

- 🐞 不具合の報告（再現条件が分かるもの）
- 💡 機能改善・拡張の提案
- 🛠 実装・修正・リファクタリング
- 📊 実験結果・再現実験の共有
- 📝 ドキュメントや設定例の改善

「コードを書かない貢献」も歓迎しています。

---

## Issue の使い分けについて

Issue は目的ごとに使い分けています。

| 種類 | 用途 |
|---|---|
| 🐛 Bug report | 明確な不具合・再現可能な問題 |
| ✨ Feature request | 機能追加・改善の要望 |
| 🛠 Contribution task | 実装・実験など具体的に着手できる作業 |

詳しくは README の **Issue 運用ルール**をご覧ください。日本語でも構いません。

---

## 作業の進め方（最小構成）

1. 対応したい Issue を選ぶ（または新規作成）
2. Fork してブランチを作成
3. 修正・実装・実験を行う
4. Pull Request（PR）を作成

PR には以下を含めてください。日本語でも構いません。

- 変更内容の概要
- 目的・背景
- （可能であれば）結果・ログ・スクリーンショット


---

## 貢献の扱いについて

貢献内容に応じて、以下の形で謝辞・記録を行います。

- リリースノートへの記載
- READMEや利用事例ページでの紹介
- 研究成果として重要な場合、論文等での謝辞・共著検討（ケースバイケース）

※ 共著を保証するものではありませんが、研究的貢献は尊重します。

---

## 相談・議論について

- 設計方針や将来構想 → Discussions
- 実装・不具合 → Issues

判断に迷う場合は Issue で相談してください。
